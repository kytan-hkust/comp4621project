# COMP 4621 Project

The proxy is implemented with C++ and tested with a Linux machine.

## Instructions

To compile the proxy, ```g++``` and ```make``` are needed.

```
make && ./proxy
```

Change proxy settings to 127.0.0.1 (localhost) as address and 12345 as port.

## Features

- [ ] Multithreading
- [x] HTTP Requests Forwarding
- [ ] HTTPS Requests Forwarding
- [x] Access Control
- [x] Caching
